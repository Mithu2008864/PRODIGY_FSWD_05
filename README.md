# Real-Time Chat Application

## Description
A real-time chat application developed using HTML, CSS, JavaScript, Node.js, Express.js and Socket.IO. Users can join chat rooms and exchange messages instantly using WebSocket technology.

## Features
- Real-time messaging
- Room-based chat
- Multiple users
- Responsive UI
- Socket.IO integration
- Express.js backend

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- Socket.IO
